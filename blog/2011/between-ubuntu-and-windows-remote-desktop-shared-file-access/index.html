1、 Ubuntu访问windows

这方面基本上Ubuntu系统上做的已经很全了，现有的应用用起来就行。远程桌面可以用Remote Desktop Viewer和Terminal Server Client两个应用。都在Applications->Internet目录下。有俩截图如下：

PS:

VNC (Virtual Network Computing)是虚拟网络计算机的缩写。VNC是一款优秀的远程控制工具软件，由著名的AT&T的欧洲研究实验室开发的。

RDP:远程桌面协议，主要用于windows 2000及更早的windows版本。 RDPv5支持xp等。

SSH: Secure Shell Protocol，是一种在不安全网络上提供安全远程登录及其它安全网络服务的协议。

共享文件的访问有另外一个应用，这个应用能进行FTP，http的访问。位于Places->connect to server。

2、windows访问ubuntu。Ubuntu互访。

公司不准私自架设ftp等服务器。所以通常通过文件共享。Linux也可以类似window那样共享文件，只是Ubuntu需要先安装应用。

Windows和Linux间共享可以用samba。Linux和Linux之间共享文件有更好的网络文件系统NFS。网上使用教程方法很多，这里就不多说了。

再废话一句，linux里很多可以用mount。
