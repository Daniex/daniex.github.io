Wordpress在页面及RSS中添加随机文章相关文章版权信息

Mar 16, 2011 · 1 minute read
DaniexWordPress编程开发
WordPress 插件 版权

1、插件法

推荐random post 和similar post 插件，属于Post-Plugin Library系列。该系列中还有recent post、 recent comment插件。

安装插件后再设置中可以设置是否在RSS和文章页面显示随机文章和相关文章列表。附带widget，可在边栏或者底部显示。如果使用代码在特定位置添加。可用<?php random-posts(); ?><?php similar-posts(); ?>嵌入。

2、代码

1）可以在single.php或者functions.php中添加代码，使显示随机文章或是版权信息。single.php的修改无法再RSS中显示。

在single.php中找到<?php the_content(); ?>代码，在其后添加代码就会在文章后显示你要的信息，可以是版权信息或是相关文章，广告等。网上有不用插件，用代码生成随机文章列表的方法，可以去搜索一下。

2）在functions.php中添加

function FUNCTIONNAME($content) { if(!is_feed() && !is_home()) { $content.= “<div class=‘sub’>“； $content.= “<a>填入想要添加的内容（版权信息广告等）</a>”; $content.= “</div>”; } return $content; } add_filter (‘the_content’, ’FUNCTIONNAME‘);
