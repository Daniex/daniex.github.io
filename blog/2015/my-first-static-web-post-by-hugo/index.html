hello hugo

Jan 22, 2015 · 1 minute read
webit
hugo hyde-X static site generator

这是我在hugo上发布的第一个post。但也不仅仅是hello world。

之前听IPN的podcast讲静态网页生成器,想到自己还有个懒得去搭理的blog，于是有有想法找时间再来折腾一下。结果一找就找了几个月。终于，花了一天时间简单过遍hugo的试用方法，解了大体的功能后，在京东云擎上搭了个试试。

要快速的建起hugo站其实只要照着他的quitstart做就可以了，还是很清楚明了的。麻烦的是定制一些自己要的功能，去修改原有的代码或者扩展他的功能。

其中比较基础的是读懂他们提供的HugoTheme然后去做修改。这里推荐其中的hyde-x，在看了几个例子和不同的theme后，我选择他入手学习。
