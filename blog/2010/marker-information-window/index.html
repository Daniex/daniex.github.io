谷歌地图多个标记（marker）和信息窗口（information window）显示

Jun 16, 2010 · 1 minute read
各种库编程开发
各种库 解决方案 问题

当谷歌地图上有多个标记（marker）和信息窗口（information window）需要显示的时候，使用基本的增加监听click的事件响应的方法，信息窗口的显示会出现问题。即在任意标记上点击，需要显示信息窗口时，窗口会固定的显示在最后画上的标记上面。

添加click事件监听方法的代码：

GEvent.addListener(marker, “click”, function() {
marker.openInfoWindowHtml(’<b>info:</b> ‘+ info);
});
map.addOverlay(marker);
查阅网上资料后，发现一个解决方案，即使用bindInfoWindowHtml方法。以上的信息显示只需修改一小部分即可：

marker = new GMarker(point);

marker.bindInfoWindowHtml(’<b>info:</b> ‘+info)

map.addOverlay(marker);
