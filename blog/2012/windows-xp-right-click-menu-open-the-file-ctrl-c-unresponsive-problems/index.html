windows xp 右键菜单、打开文件、ctrl+c 奇慢问题

Feb 23, 2012 · 1 minute read
Daniex便笺生活点滴
windows 问题

办公电脑出现这样的问题有半个月，一直没去折腾他。今儿网上找了下，发现方法很多，不知那个好。症状最符合的就是如下一篇，照做，立马搞定。

————————————————————————————————————————

直接症状：

直接双击打开一个文件，比如一个 txt/Word 文档，要等超过15秒的时间，如果先打开 Word，然后再把文档拖到 Word 中则正常。 在某文件上点击右键，弹出右键菜单需要超过15秒的时间。 对某个文件进行键盘上的 Ctrl+C, Ctrl+V 操作，超过15秒才有反应。。。

间接症状：

断掉网络，如拔掉网线后，双击打开文件的速度和右键菜单的速度就都正常了。

解决方法：

点击“开始”->“运行”。

在“打开”后输入“regedit”，打开注册表。

先备份注册表，以防万一。方法是，在注册表编辑器中，选择“文件”->“导出”，将当前注册表导出到一个安全的位置。

回到注册表顶端，打开“我的电脑”->“HKEY_CLASSES_ROOT”->“*”->“shellex”

将下面的两个子项改个名字，如改成“ContextMenuHandlers-”和“PropertySheetHandlers-”

然后关闭注册表编辑器。

天下太平。
